Selected Work

Case studies & impact

Challenge

Enterprise Cloud Migration

Legacy on-premises workloads limited scalability. Manual provisioning led to 3–5 day release cycles, frequent deployment outages, and zero DR capability.

Solution

Architected lift-and-shift then re-platform strategy across AWS & Azure. Introduced Terraform modules for repeatable infra, multi-AZ deployments, and blue-green CI/CD via GitHub Actions. Built centralised observability with Prometheus + Grafana.

4× Release frequency
99.95% uptime post-migration
Challenge

Kubernetes Platform Engineering

Microservices teams managed ad-hoc deployments. No standardisation, no SLOs, 300+ noisy alerts, and MTTR averaging 4+ hours.

Solution

Built an internal developer platform on AKS/EKS with Helm charts, ArgoCD GitOps, and a self-service portal. Defined per-service SLOs. Consolidated monitoring into a unified Grafana dashboard with actionable alerting and on-call runbooks.

60% MTTR reduction
On-call load halved
Challenge

Cloud Cost Optimisation

Uncontrolled cloud sprawl resulted in 40% of spend on idle or over-provisioned resources. No tagging strategy, no budget alerts, no team accountability.

Solution

Implemented FinOps practices: rightsizing analysis, reserved instance purchasing, automated non-prod shutdown after hours, and per-team cost dashboards. Mandatory tagging enforced via Terraform.

35% Cloud cost reduction
Within 6 months
