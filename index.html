Case Studies

Selected work & impact

Challenge

Enterprise Cloud Migration

Legacy on-premises workloads limited scalability. Manual provisioning led to 3–5 day release cycles, frequent outages during deployments, and no DR capability.

Solution

Architected lift-and-shift then re-platform strategy across AWS & Azure. Introduced Terraform modules for repeatable infra, multi-AZ deployments, and blue-green CI/CD pipelines via GitHub Actions. Built centralised observability with Prometheus + Grafana.

4×
Release frequency; 99.95% uptime post-migration
Challenge

Kubernetes Platform Engineering

Microservices teams each managed their own ad-hoc deployments. No standardisation, no SLOs, alert fatigue from 300+ noisy monitors, and MTTR averaging 4+ hours.

Solution

Built an internal developer platform on AKS/EKS with Helm charts, ArgoCD GitOps, and a self-service portal. Defined SLOs per service. Consolidated monitoring into a unified Grafana dashboard with actionable alerting and on-call runbooks.

60%
Reduction in MTTR; on-call load halved
Challenge

Cloud Cost Optimisation

Uncontrolled cloud sprawl resulted in 40% of cloud spend on idle or over-provisioned resources. No tagging strategy, no budget alerts, no accountability per team.

Solution

Implemented FinOps practices: rightsizing analysis, reserved instance purchasing, automated shutdown of non-prod environments after hours, and per-team cost dashboards. Introduced mandatory tagging policy enforced via Terraform.

35%
Cloud cost reduction within 6 months
